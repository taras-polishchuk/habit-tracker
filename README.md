# Habit Tracker

A minimalist daily habit tracker built with React. Add habits, check them off each day, and stay consistent.

**Live demo:** https://taras-polishchuk.github.io/habit-tracker/

## Features

- Add and delete custom habits
- Check off habits for today with a checkbox
- Progress counter (e.g. "3 / 5 done today")
- Celebration banner when all habits are completed
- Persists data in `localStorage` — survives page refresh

## Tech stack

- React 18
- Vite
- CSS (no UI libraries)
- GitHub Actions + GitHub Pages for deployment

## Run locally

```bash
npm install
npm run dev
```
